#### Manipulador de XML do Padrão Nacional
<p> O objetivo deste simples projeto é demonstrar as operações que envolve o XML da DPS no processo de comunicação junto ao Padrão Nacional</p>
<p> Com ele é possível:</p>
<ul>
  <li>Compactar e codificar em base 64 um arquivo XML de um DPS assinado gerando no final um JSON semelhante ao enviado no payload da requisição a API do Padrão Nacional</li>
  <li>Descompactar e descodificar o valor do JSON que é enviado no payload da requisição a API do Padrão Nacional para obter o XML da DPS assinado</li>
</ul>
